== Description ==

-- WordPress news plugin is the free version of the plugin WP Latest posts which found right here with a lot of information.
-- World news Plugin has used third-party API https://newsapi.org/ to load content from another website.

= Rules Document =

*   Show Latest posts instances in your site.
*   Display as a sidebar widget.
*   Display Content title, image, publish date and description.

== Installation ==

This section describes how to install the plugin and get it working.

1. Navigate to ‘Add New’ under ‘Plugins’ menu in admin area.
2. Upload "wp-world-news.zip" from your downloaded directory or choose from WordPress Plugins repository.
3. Activate the plugin through the 'Plugins' screen in WordPress.
4. Go to the Appearance->Widgets.You can see widget "World news".Drag and drop to the desired area.
